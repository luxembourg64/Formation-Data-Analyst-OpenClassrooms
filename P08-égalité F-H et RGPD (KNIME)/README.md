# P08 – Indicateurs d'égalité Femmes/Hommes avec KNIME

> **Formation :** Data Analyst – OpenClassrooms | **Outil :** KNIME

## 🎯 Objectifs du projet

En tant que Data Analyst dans un cabinet de conseil en transformation digitale, automatiser la création d'un rapport sur l'**égalité professionnelle Femmes/Hommes** à partir des données du SIRH, en respectant le RGPD.

**Contexte :** Toute entreprise de +50 salariés doit publier chaque année son index d'égalité F/H avant le 1er mars. Le projet vise à automatiser ce diagnostic via un workflow KNIME.

- Collecter et agréger des données issues du SIRH
- Créer un workflow KNIME pour générer automatiquement les graphiques du diagnostic
- Produire un fichier CSV prêt pour des analyses complémentaires (Tableau Software)
- Vérifier la cohérence et la fiabilité des données en conformité RGPD
