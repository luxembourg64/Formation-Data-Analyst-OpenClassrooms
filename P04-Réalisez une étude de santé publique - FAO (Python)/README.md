# P04 – Étude de santé publique avec Python (FAO)

> **Formation :** Data Analyst – OpenClassrooms | **Durée estimée :** 80 h | **Outil :** Python

## 🎯 Objectifs du projet

En tant que Data Analyst au sein de la **Food and Agriculture Organization of the United Nations (FAO)**, réaliser une étude de santé publique à partir de données mondiales sur la nutrition et la sous-alimentation.

- Premier projet de programmation du parcours (Python / Notebook Jupyter)
- Explorer et analyser des jeux de données réels de la FAO
- Identifier des corrélations et tendances sur la malnutrition mondiale
- Présenter les résultats lors d'une soutenance

## 📋 Compétences visées

| Compétence | Description |
|---|---|
| Analyse de données | Explorer et analyser des données avec Python |
| Notebook | Travailler avec Jupyter Notebook |
| Visualisation | Produire des graphiques analytiques |
