# P06 – Optimisation de la gestion des données d'une boutique

> **Formation :** Data Analyst – OpenClassrooms | **Durée estimée :** 70 h | **Outil :** Python

## 🎯 Objectifs du projet

Mission professionnelle : préparer, nettoyer et analyser les données de stock d'une boutique pour en extraire des insights exploitables.

- Utiliser des librairies Python (pandas, matplotlib, etc.)
- Réaliser une analyse exploratoire de données (EDA)
- Effectuer des jointures entre fichiers de données
- Produire des analyses univariées et multivariées
- Traiter les données en conformité avec le RGPD

## 📋 Compétences visées

| Compétence | Description |
|---|---|
| Prétraitement | Traiter et nettoyer des données, gérer les incohérences |
| EDA | Effectuer des analyses univariées et multivariées |
| RGPD | Définir la gestion des données en conformité réglementaire |
