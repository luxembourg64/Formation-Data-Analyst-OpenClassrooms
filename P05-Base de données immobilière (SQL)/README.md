# P05 – Base de données immobilière avec SQL

> **Formation :** Data Analyst – OpenClassrooms | **Durée estimée :** 60 h | **Outil :** SQL

## 🎯 Objectifs du projet

Conception et exploitation d'une base de données immobilière complète, en respectant les règles de modélisation relationnelle (3NF) et les exigences du RGPD.

- Analyser des données brutes pour extraire les variables clés
- Concevoir un schéma relationnel en 3ème forme normale (3NF)
- Créer les tables SQL et importer les données nettoyées
- Rédiger des requêtes avancées : agrégations, filtres, jointures multiples
- Analyser le marché immobilier et produire des indicateurs clés

## 📋 Compétences visées

| Compétence | Description |
|---|---|
| Modélisation | Concevoir un schéma relationnel 3NF et un dictionnaire de données |
| Création BDD | Créer des tables, importer des CSV, gérer les contraintes |
| Requêtage avancé | Agrégations, jointures, filtres complexes |
| RGPD | Assurer la conformité des données personnelles |
