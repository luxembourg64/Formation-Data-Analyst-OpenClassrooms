# P12 – Détection de faux billets avec Python et Machine Learning

> **Formation :** Data Analyst – OpenClassrooms | **Durée estimée :** 70 h | **Outils :** Python, Machine Learning

## 🎯 Objectifs du projet

À partir d'un jeu de données réel sur les dimensions physiques de vrais et faux billets de banque, construire un **modèle de classification** capable de détecter automatiquement les contrefaçons.

Première mission complète de **modélisation prédictive** du parcours — un excellent tremplin vers la Data Science.

- Explorer et visualiser les données (dimensions des billets)
- Préparer et nettoyer le jeu de données
- Construire et comparer plusieurs modèles de classification
- Évaluer leurs performances et sélectionner le plus efficace
- Identifier les critères discriminants entre vrais et faux billets
- Présenter les résultats et le modèle de façon claire

## 📋 Compétences visées

| Compétence | Description |
|---|---|
| ML supervisé | Utiliser un modèle supervisé pour prédire un phénomène |
| ML non supervisé | Entraîner et exploiter un modèle non supervisé |
| Évaluation | Comparer des modèles et sélectionner le plus performant |
