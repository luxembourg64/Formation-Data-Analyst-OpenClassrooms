# P07 – Dashboard de suivi de projets avec Power BI

> **Formation :** Data Analyst – OpenClassrooms | **Durée estimée :** 60 h | **Outil :** Power BI

## 🎯 Objectifs du projet

En tant que consultant Data Analyst chez une ESN Data, créer un tableau de bord dynamique pour visualiser l'avancement de projets clients, de la collecte du besoin à la livraison.

- Identifier les enjeux métier du client via un **Product Strategy Canvas**
- Collecter les user stories et définir les indicateurs clés
- Préparer et transformer les données dans Power BI
- Développer des visualisations interactives
- Présenter le dashboard au client lors d'une soutenance

## 📋 Compétences visées

| Compétence | Description |
|---|---|
| BI / Dashboard | Concevoir un tableau de bord interactif avec Power BI |
| Analyse métier | Comprendre le besoin client avant l'implémentation |
| Data visualisation | Produire un reporting pour faciliter la prise de décision |
