# P10 – Étude sur l'accès à l'eau potable dans le monde

> **Formation :** Data Analyst – OpenClassrooms | **Durée estimée :** 70 h | **Outil :** Power BI

## 🎯 Objectifs du projet

En tant que consultant Data Analyst en mission dans une ONG, concevoir un tableau de bord interactif pour analyser l'accès à l'eau potable à l'échelle mondiale et orienter les décisions stratégiques.

- Rechercher et sélectionner des données pertinentes sur l'eau potable
- Pré-traiter et manipuler les données avec Power BI
- Appliquer les bonnes pratiques de data storytelling
- Créer un dashboard interactif pour engager un public non technique

## 📋 Compétences visées

| Compétence | Description |
|---|---|
| Storytelling | Proposer un récit des résultats dynamique pour engager un public |
| Visualisation | Optimiser une solution accessible adaptée au public et aux données |
| Dashboard | Créer un tableau de bord interactif disponible et accessible |
