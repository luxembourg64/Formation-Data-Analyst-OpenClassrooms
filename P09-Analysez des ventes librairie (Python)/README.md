# P09 – Analyse des ventes d'une librairie avec Python

> **Formation :** Data Analyst – OpenClassrooms | **Durée estimée :** 90 h | **Outil :** Python

## 🎯 Objectifs du projet

Analyser les données de ventes d'une librairie pour identifier des tendances, tester des hypothèses et produire des insights actionnables à partir de séries temporelles.

- Analyser des indicateurs de ventes et des corrélations
- Réaliser des statistiques descriptives approfondies
- Produire des visualisations avec les librairies Python
- Analyser des séries temporelles pour mesurer des phénomènes statistiques
- Réaliser des tests statistiques pour valider des hypothèses

## 📋 Compétences visées

| Compétence | Description |
|---|---|
| Séries temporelles | Analyser un jeu de données avec séries temporelles |
| Tests statistiques | Réaliser des tests pour valider des hypothèses |
| Visualisation | Générer des graphiques analytiques avec Python |
