# P03 – Requêtes SQL sur une base de données Assurances

> **Formation :** Data Analyst – OpenClassrooms | **Durée estimée :** 30 h | **Outil :** SQL

## 🎯 Objectifs du projet

Projet d'entraînement aux fondamentaux du SQL et de la gestion de bases de données, sur un jeu de données issu du secteur des assurances.

- Comprendre les enjeux de la gestion des données
- Créer et structurer un schéma de base de données
- Charger des données dans la base
- Rédiger des requêtes SQL pour aider à la prise de décision métier

## 📋 Compétences visées

| Compétence | Description |
|---|---|
| Modélisation | Créer un schéma de base de données |
| Chargement | Charger des données dans une base |
| Requêtage | Rédiger des scripts SQL pour l'aide à la décision |
