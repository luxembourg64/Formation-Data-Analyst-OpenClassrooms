# P13 – Portfolio professionnel de la data

> **Formation :** Data Analyst – OpenClassrooms | **Projet final**

## 🎯 Objectifs du projet

Projet de clôture du parcours : construire un **portfolio en ligne personnalisé** pour valoriser les compétences acquises et optimiser son employabilité en tant que Data Analyst professionnel.

- Formaliser et mettre en valeur les compétences développées tout au long de la formation
- Rédiger un cahier des charges fonctionnel pour cadrer les besoins
- Adopter une posture de consultant Data Analyst
- Effectuer une veille métier et technologique
- Organiser le projet d'analyse data avec des outils de gestion de projet

## 📋 Compétences visées

| Compétence | Description |
|---|---|
| Communication | Adapter sa posture en se positionnant comme consultant |
| Veille | Effectuer une veille métier et technologique |
| Documentation | Formaliser les procédures pour assurer une gestion fiable |
| Gestion de projet | Organiser un projet data avec des outils dédiés |
| Cahier des charges | Formaliser un CDC fonctionnel pour cadrer besoins et solutions |
