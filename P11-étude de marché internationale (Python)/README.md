# P11 – Étude de marché internationale avec Python

> **Formation :** Data Analyst – OpenClassrooms | **Durée estimée :** 90 h | **Outil :** Python

## 🎯 Objectifs du projet

Endosser la posture de Data Analyst conseil auprès de la direction : réaliser une **étude de marché internationale** complète, de la collecte des données jusqu'aux recommandations stratégiques, en utilisant des techniques avancées de réduction de dimension et de clustering.

- Collecter, nettoyer et préparer les données de marché
- Réaliser des analyses multivariées (ACP, clustering)
- Réduire la dimension du jeu de données (PCA)
- Sélectionner un cluster pertinent et interpréter les résultats
- Formuler des recommandations stratégiques adaptées à un auditoire non technique

## 📋 Compétences visées

| Compétence | Description |
|---|---|
| Analyse multivariée | Comprendre la corrélation entre les variables |
| Réduction de dimension | Optimiser le jeu de données (PCA) |
| Machine Learning | Exploiter un modèle d'apprentissage non supervisé (clustering) |
| Communication | Adapter le discours à un auditoire non technique |
